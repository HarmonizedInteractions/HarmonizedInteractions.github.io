# SPEC - 技術仕様・要件定義

## サイト概要
株式会社Harmonized Interactionsのコーポレートサイト。GitHub Pagesで公開し、カスタムドメイン（harmonized-interactions.com）で運用する。

## 機能要件

### ページ構成
1. **HOME（トップページ）** - ヒーローセクション、会社のキャッチコピー、事業概要サマリー
2. **Company（会社概要）** - 会社情報テーブル、メンバー紹介（プロフィールカード形式）
3. **Business（事業内容）** - 6つの事業領域をカード/セクション形式で紹介
4. **Technology（対応ソフトウェア・技術）** - 対応ソフトウェア一覧、技術スタックの紹介
5. **Contact（連絡先）** - 連絡先情報

### 多言語対応
- 日本語（ja）と英語（en）の2言語
- ヘッダーに言語切替ボタンを配置
- 実装方式：ディレクトリ方式（`/` = 日本語、`/en/` = 英語）

### ナビゲーション
- レスポンシブハンバーガーメニュー（モバイル）
- 固定ヘッダーにロゴ＋ナビゲーション＋言語切替

## 非機能要件
- レスポンシブデザイン（モバイル・タブレット・デスクトップ）
- GitHub Pages対応（静的サイト）
- カスタムドメイン対応（harmonized-interactions.com）
- パフォーマンス：Lighthouse 90+
- アクセシビリティ：セマンティックHTML

## 技術構成
- **ホスティング**: GitHub Pages（HarmonizedInteractions.github.io）
- **ビルドツール**: なし（静的HTML/CSS/JS）または軽量SSG
- **CSS**: モダンCSS（CSS変数、Flexbox、Grid）
- **JavaScript**: バニラJS（フレームワーク不要）
- **フォント**: Noto Sans JP + Noto Sans

## デザイン仕様

### カラーパレット（ロゴ基準）
- プライマリ（ダークブルー）: #1A5F8B
- セカンダリ（シアン）: #3AAFDA
- テキスト（グレー）: #555555
- 背景（白）: #FFFFFF
- 背景（ライトグレー）: #F5F7FA

### タイポグラフィ
- 見出し: Noto Sans JP Bold
- 本文: Noto Sans JP Regular
- 英語テキスト: Noto Sans
